# VIVARIA

This study introduces **VIVARIA** (*Desktop Hand-Tracked Virtual Laboratory*), a desktop simulator built in Unity designed for preliminary training in laboratory procedures involving small rodents, fully aligned with the 3Rs ethical framework (*Replacement, Reduction, Refinement*). Utilizing an Ultraleap 3Di markerless optical sensor for hand tracking at 90 Hz, the system implements a modular *task/step/condition* architecture that tracks objective interaction metrics in real time and enforces an automatic state rewind upon critical user errors.

Initial validation involved 26 students from veterinary medicine and biotechnology across three tasks of increasing complexity: cage transfer, maxillary vein blood sampling, and intraperitoneal injection. The results closely reflect real-world procedural difficulties: while cage transfer was immediate ($27.99\pm15.05$ s, zero errors), the maxillary sample required significantly more time ($124.72\pm76.09$ s) and registered the lowest success rate (46.15%), concentrating most tracking anomalies and regrasp attempts. Subjective evaluation via questionnaires confirmed strong educational value (mean usefulness score of 4.46/5), though highlighting room for improvement in fine-motor tracking stability. Overall, VIVARIA proves to be a scalable, effective preparatory tool to consolidate procedural workflows prior to *in vivo* animal training.
